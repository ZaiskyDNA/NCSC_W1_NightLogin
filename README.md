# NCSC NL Training Week 1: General Skills

Repositori ini berisi materi dan *write-up* tantangan CTF kategori **General Skills** untuk pembelajaran minggu pertama komunitas NCSC NL.

## Materi Pembelajaran
Fokus pada minggu ini adalah penguasaan fondasi dasar sebelum masuk ke kategori yang lebih spesifik, meliputi:
- Perintah dasar Linux (CLI).
- Penggunaan SSH untuk koneksi ke server soal.
- Manipulasi dan ekstraksi file (seperti penggabungan *file* dan *unzip*).

## Cara Menggunakan Repositori Ini
1. Buka folder tantangan yang tersedia.
2. Usahakan untuk menyelesaikan soal secara mandiri terlebih dahulu di terminal masing-masing.
3. Jika mengalami kendala, silakan baca file `writeup.md` di dalam folder tersebut sebagai panduan langkah demi langkah.

Selamat belajar dan berlatih!
